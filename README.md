# omnifood
# Description:
This is a responsive food delivery landing page built using HTML, CSS, and a simple grid system.It includes sections like features, meals showcase, how it works, cities, testimonials, pricing plans, and a contact form. I also used AOS for animations and practiced layout techniques like floats and clearfix.
This project helped me strengthen my front-end fundamentals, especially in building responsive layouts from scratch without frameworks.
